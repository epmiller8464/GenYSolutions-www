A Civilized Company Coding In Uncivilized Times

There’s no doubt that life and business have gotten more complex, even as certain tasks and activities have become easier due to information technology. The pace of change is quickening. The global economy is becoming still more connected, creating a much larger and more diverse population of customers and suppliers. Manufacturing and services are increasingly targeted at smaller, specialized markets due to the flexibility that IT provides in these areas. The 3D printing revolution is a perfect example. We know from our knowledge of the patterns of evolution that, in reality, systems tend to become more complex as they evolve, then become simplified again. The problem is how to develop better systems designed to fit your business models, processes, products and services in a way that minimizes unnecessary complexity and provides true business value. Many commercial software products often claim to provide an all-inclusive “out-of-the-box” solution that satisfies the unique requirements for a wide-spectrum of organizations in an even broader range of industries. Often leaving the client with a large bill, unfulfilled requirements, and disgruntled stakeholders. At Gen Y Solutions we know that software is not a one­ size fits all answer for all the diverse issues an organization may face today. We believe that bringing your problem full circle will allow us to engineer the best system to meet your needs.Our sole mission is to develop software that meets your needs, budget, and timeline.

Development process

The goal of our software creation process is to allow us to meet our client's goals the best we can, while keeping good quality of code practices and meeting deadlines. We modify the process based on specific client's needs.

The main steps are:

  • Setup

    This includes: Tasks management system - thanks to this system, our clients have constant access to information about the project's progress and are able to comment on particular tasks. A secure system for storing sensitive information Knowledge base Dropbox - a directory that features cloud based file sharing GIT - a code versioning tool, which allows us to keep history of changes and allows multiple developers to cooperate on the project. We also use it for deployment scripts. Bitbucket - a tool that allow for code reviews and collaboration on the code

  • Consultation

    In this phase we use our experience to suggest modifications and improvements for the system. Our goal is to lower the amount of work required, increase stability and performance, as well as improve user experience.

  • Mockups

    In this phase we closely cooperate with a client. We create sketches of all system pages and plan out the user interface. The goal of this phase is to eliminate any misunderstandings, and create the interface that is best suited for our client needs. This eliminates extra work of modifying a live system

  • Technical Specifications

    Based on layouts, we create a detailed plan for the implementation of features. This phase doesn't require very close cooperation with a client. All technical specifications are available in Basecamp

  • Implementation

    In this phase developers create features based on the documentation prepared in previous steps. This phase is usually separated into milestones, each with a specific deadline. With completion of every milestone we consult the project with a client to make sure our solutions meet all requirements. Based on the client's remarks, we can apply modifications to existing functionalities in the following milestones. During this phase we use: Deployment scripts - they allow us to update code on the server with minimal downtime and with the possibility of automatic reverts Unit tests and code standard checkers - they ensure that all components of the system are of good quality Autmatic documentation tools - they make sure it's easy to bring new people into a development.

  • QA Process

    After a project is complete, we test the system and apply bugfixes. This process can be run by a client, or on request by our team.

  • Graphic Designs

    This phase is done in close cooperation with our client. Our graphic designer creates the graphic design based on layouts and remarks that he receives from a client. Afterwards, graphics are consulted with a client and modified according to client's remarks.

  • Deployment (Hosting and Support)

    If needed, we can provide hosting and support for the existing system.

  • Warranty

    We give lifetime warranty for all critical errors of the system. On request we also offer a warranty package for modification of functionalists that are exceeding the specification.
